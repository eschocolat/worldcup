# WORLD CUP 2026 · 우승 예측 분석

Opta 슈퍼컴퓨터 · 베팅시장(06.18) · FIFA 랭킹(06.11) · Elo 평가 + 1차전 결과를 종합한
**월드컵 2026 우승 예측**을 인포그래픽으로 보여주는 정적 사이트.

**결론: 우승 예상 1위 — 🇫🇷 프랑스 (추정 17%)**

## 섹션

1. **Hero** — 결론(프랑스 17%) + 핵심 근거 수치
2. **왜 프랑스인가** — 4가지 근거 (모델 / 시장 / 음바페 폼 / 경쟁자 흔들림)
3. **우승 확률 상위권** — 스크롤 시 채워지는 막대 차트 (Top 10)
4. **예상 조별리그** — 48개국 12개 조 카드 (1·2·3위 진출/탈락)
5. **토너먼트 브래킷** — 32강 → 결승 가로 스크롤 트리
6. **결승 예측** — 프랑스 2-1 아르헨티나 + 최종 예상 순위
7. **대한민국** — 16강 예상 경로
8. **가장 큰 변수** — 5개 팀별 리스크

## 디자인

**Dark OLED Broadcast** — 순흑 배경, 시안/앰버 네온 액센트, 필름 그레인,
스포츠 중계 그래픽 무드. 폰트: Anton · Black Han Sans · Archivo Expanded · Spline Sans Mono.

## 실행

빌드 불필요. 정적 파일이라 그냥 열면 됩니다.

```bash
# 브라우저로 바로 열기
open index.html

# 또는 로컬 서버 (권장 — file:// 폰트/캐시 이슈 회피)
python3 -m http.server 8080
# → http://localhost:8080
```

## 구조

```
index.html         # 마크업 (시맨틱 섹션)
assets/styles.css  # Dark OLED Broadcast 디자인 시스템 (CSS 변수)
assets/app.js      # 데이터 + 차트/조별/브래킷 렌더링, 스크롤 애니메이션
```

데이터는 모두 `assets/app.js` 상단의 `ODDS` · `GROUPS` · `ROUNDS` 상수에 있어
값만 바꾸면 차트·표·브래킷이 자동으로 다시 그려집니다.

## 접근성 / 성능

- 시맨틱 HTML, 키보드 포커스 스타일, `prefers-reduced-motion` 대응
- 외부 JS 프레임워크 0개 · 이미지 0개 (이모지 국기 + CSS만)
- 반응형 (데스크탑 → 모바일 390px까지 검증)
